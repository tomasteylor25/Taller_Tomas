# Mi Portafolio Personal - Taller de Git

Este proyecto consiste en el desarrollo de un portafolio personal utilizando **HTML** y **CSS**, aplicando un flujo de trabajo mediante **Git** y **GitHub**.

## Tecnologías Utilizadas
* **HTML**: Estructura del contenido.
* **CSS**: Diseño visual y estilos responsivos.
* **Git**: Control de versiones.
* **GitHub**: Alojamiento del repositorio y gestión de tareas (Issues/PRs).

## Flujo de Trabajo Git
En este taller se aplicó un flujo de trabajo basado en ramas:
1. `main`: Rama de producción (versión final).
2. `develop`: Rama de integración.
3. `feature/`: Rama para el desarrollo de funcionalidades específicas.

Se utilizaron **Issues** para planificar las tareas y **Pull Requests (PR)** para la revisión y fusión de código hacia la rama `develop`.

## Estructura del Proyecto
* `/index.html`: Página principal.
* `/styles.css`: Hoja de estilos.
* `/asset/`: Carpeta para imágenes y recursos multimedia.
